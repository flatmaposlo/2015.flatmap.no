INTRODUCTION TO HASKELL

workshop

This workshop will teach you the basics of the purely functional programming language Haskell. No prior experience with Haskell or functional programming language is required.

We will start by looking at what distinguishes Haskell from other programming languages – including other functional programming languages – namely its rich type system, lazy evaluation and no mutable state. We will also look at how, with Haskell's powerful type inference, we get strong correctness guarantees without having to sprinkle type annotations everywhere.

We will then dive into some code and see how to use Haskell in practice. You will be using using some of the built-in data types, and you will define our own special-purpose data types to solve problems in a test-driven fashion.

After having dipped our toes in some Haskell code, we will end by talking a bit about how not allowing side effects makes Haskell great for concurrency and parallelism and look a bit at how Haskell is used in practice in Industry.

We are going to use FPComplete's web-IDE for the coding part, so you do not need to install anything beforehand. You are of course free to use your favourite text editor, in which case you should install the Haskell compiler GHC and the build tool `cabal-install` beforehand. (Your editor probably already has a Haskell mode.)

After a brief return to his former passion of low-level programming, working on drivers at ARM, his nascent love for Haskell and functional programming sprung into full bloom during a research internship at Galois.
